# Student Management System

## Description
A Python-based Student Management System that allows users to manage student records using JSON.

## Features
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student

## Technologies Used
- Python
- JSON
- Git
- GitHub

## How to Run
1. Open the project folder.
2. Run:
   python main.py
